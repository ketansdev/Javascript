## Overview

An OTP Verification system built using HTML, CSS, and JavaScript that simulates a real-world authentication flow.

When the user clicks **Get OTP**, a randomly generated OTP is displayed via an alert for demo purposes. The user then verifies it through a popup-based input interface.  

This project focuses on user interaction, input handling, DOM manipulation, and modal-based UI design.

---



---

## 🛠 Tech Stack  
- HTML  
- CSS  
- JavaScript (Vanilla JS)  
- DOM Manipulation  

---

## ✨ Features  
- Generates a random OTP on button click  
- Displays OTP via alert for demonstration & testing  
- Popup (modal) based OTP input UI  
- Auto-focus and smooth navigation between OTP input fields  
- Validates entered OTP against generated value  
- Clear success and error messages  
- Clean, minimal & responsive design  
- Beginner-friendly frontend authentication simulation
