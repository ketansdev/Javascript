## 📌 Overview  

A **User Profile Card Generator** built using HTML, CSS, and JavaScript that allows users to create a personalized profile card by entering basic details.

The project dynamically generates a visually styled profile card from user input, demonstrating interactive form handling and real-time UI rendering without reloading the page.

It focuses on DOM manipulation, form validation, and dynamic content creation.

---

## 🛠 Tech Stack  

- HTML  
- CSS  
- JavaScript (Vanilla JS)  
- DOM Manipulation  

---

## ✨ Key Features  

- Create a profile card using user-provided details  
- Supports profile image via image URL with fallback image  
- Dynamically updates UI without page reload  
- Validates required input fields before generation  
- Clean, responsive, and reusable card layout  
- Beginner-friendly project showcasing interactive UI logic  

---

## 🧠 What I Learned  

- Handling form inputs using JavaScript  
- Validating required fields before rendering content  
- Dynamically creating and updating DOM elements  
- Working with image URLs and fallback handling  
- Writing clean and reusable UI generation logic  
- Improving layout structure using CSS
