## 📌 Overview  

A **GitHub User Finder** built using HTML, CSS, and JavaScript that allows users to search for GitHub profiles by username.

The application fetches real-time data from the **GitHub Public API** and displays user details such as profile information, repositories count, and followers in a clean interface.

This project demonstrates how to integrate external APIs, handle asynchronous data fetching, and dynamically render user information on the webpage.

---

## 🛠 Tech Stack  

- HTML  
- CSS  
- JavaScript (Vanilla JS)  
- GitHub Public API  
- Fetch API  

---

## ✨ Key Features  

- Search GitHub users by username  
- Fetches real-time data using the GitHub Public API  
- Displays user avatar, name, bio, and profile link  
- Shows key statistics like repositories, followers, and following  
- Handles invalid usernames with clear error messages  
- Clean, responsive, and user-friendly design  
- No API key required  

---

## 🧠 What I Learned  

- Working with external APIs using the `fetch()` method  
- Handling asynchronous JavaScript using promises  
- Processing JSON responses from APIs  
- Dynamically rendering API data in the DOM  
- Managing error states when users are not found  
- Building interactive search-based UI components
